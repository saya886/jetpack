# 项目概览

## 背景

Jetpack 是一个横板跳跃闯关网页游戏。用户通过网页登录后进入游戏，可闯关并在商店购买人物皮肤。

## 当前阶段

项目初始化阶段。

## 已配置范围

- Vue 客户端基础工程。
- FastAPI 后端基础工程。
- OpenAPI 基础契约。
- AI 协作入口与项目规范。
- Issue 与 PR 模板。
- 美术资源目录。

## 暂不包含

- 游戏关卡玩法。
- 注册、登录、鉴权实现。
- 商店购买逻辑。
- 角色皮肤资源。
- 数据库模型和部署流水线。

## 后续建议拆分

- 待排期-后端-用户注册登录接口
- 待排期-客户端-用户注册登录页面
- 待排期-客户端-横板关卡原型
- 待排期-后端-皮肤商店接口
- 待排期-美术-主角基础皮肤
