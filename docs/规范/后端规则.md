# 后端规则

后端位于 `server/`，使用 Python、FastAPI、uv 和 pytest。

## 启动

```bash
cd server
uv sync
uv run fastapi dev src/jetpack_api/main.py
```

Windows 后台或控制台编码异常时可使用：

```bash
uv run uvicorn jetpack_api.main:app --reload --app-dir src
```

## 测试

```bash
cd server
uv run pytest
```

## 代码约定

- API 路由保持清晰的版本前缀，例如 `/api/v1`。
- 请求和响应模型应使用 Pydantic。
- 新增或修改接口时同步更新 `server/openapi/openapi.yaml`。
- 数据库、鉴权、支付等高风险修改必须在 PR 中说明影响范围、风险和回滚方式。
