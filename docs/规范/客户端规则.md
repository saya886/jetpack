# 客户端规则

客户端位于 `client/`，使用 Vue 3、Vite、TypeScript、Pinia 和 Vue Router。

## 启动

```bash
cd client
npm install
npm run dev
```

## 质量检查

```bash
npm run lint
npm run type-check
npm run test:unit
npm run build
```

## 接口调用

接口调用链路：

```text
server/openapi/openapi.yaml -> client/api-client -> client/src
```

客户端不得绕过接口契约直接猜测字段。接口变化必须同步更新 OpenAPI。

## 美术资源

使用美术资源前优先查看：

1. 对应 Issue
2. `docs/功能`
3. `art/`

资源版本不明确时，应在 Issue 中确认。
